Here are the flow steps for the process:

1. **Run the Application**:  
   - Execute the command: `streamlit run login.py` to start the application.

2. **Login**:  
   - On the login screen, enter the credentials:  
     - **Username**: `admin`  
     - **Password**: `admin`  
   - Click the **Login** button to proceed.

3. **Navigate to Home Page**:  
   - After successful login, the home page is displayed.  
   - Click the **Test Case Generation** button to navigate to the Test Case Generation page.

4. **Upload User Story File**:  
   - On the Test Case Generation page, upload a file named `user_story.xlsx` in the **Upload User Story File** section.

5. **Upload TDM File**:  
   - Upload a file named `story_id_tdm.xlsx` in the **Upload TDM File** section.

6. **Generate Test Cases**:  
   - Click the **Generate Test Cases** button.  
   - After processing, click the **Download Test Cases** button to download a ZIP file containing the test case step files.

7. **Navigate to Feature Generation**:  
   - Click the **Continue to Feature Generation** button to navigate to the Feature Generation page.

8. **Generate Feature Files**:  
   - On the Feature Generation page, click the **Generate Feature File** button to process the feature files.  
   - After processing, click the **Download Feature Files** button to download a ZIP file containing the feature files.

9. **Logout**:  
   - Click the **Logout** button to log out of the application.
